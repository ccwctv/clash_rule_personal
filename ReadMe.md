

# 策略组位置对应

- 一一对应
- 顺序也要对应



# 注意规则顺序

- 规则越靠前，优先级越高
- 越特殊的规则，越要放前面
- 例如：OneDrive 策略组，要排在Microsoft 策略组之前



# 规则集

- 注意规则集类型
  - text 文件：classical
  - mrs 文件：domain
- Github 上文件，点击右上角 Raw 打开，才是文件地址
- Github 地址前，添加加速地址，以防拉取配置失败
